# Omega Admin

Modern responsive SaaS Admin Dashboard built using React + Vite.

## Features

- Modern SaaS UI
- Responsive Sidebar
- Product Management
- Product Details
- Analytics Dashboard
- Search / Filter / Sorting
- Pagination
- Toast Notifications
- Mobile Responsive
- Recharts Integration
- Framer Motion Animations
- Real-time Polling
- Column Toggle System

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router DOM
- Axios
- Recharts
- Framer Motion
- React Hot Toast

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deploy easily using Vercel.

## Environment Variables

Create `.env`

```env
VITE_API_URL=https://dummyjson.com
```
